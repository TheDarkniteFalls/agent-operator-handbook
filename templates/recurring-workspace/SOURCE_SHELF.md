# Source Shelf

This shelf names the evidence the recurring workflow is allowed to trust.

Use the smallest source set that can answer the workflow safely. Add a source
because it has a clear purpose, not merely because it is available.

## Source Health Rules

- `source_backed`: the named source directly supports the claim.
- `provisional`: useful interpretation that is not yet durable truth.
- `unknown_due_to_missing_source`: the required evidence is unavailable.
- `conflict`: authoritative sources disagree.
- `review_required`: consequence or ambiguity requires human judgement.

A search hit, filename, meeting attendance, or repeated mention may help find
evidence. It does not by itself prove ownership, a relationship, provenance, or
completion.
